# Генератор и анализатор данных в Docker

## Описание

Проект состоит из двух Docker-контейнеров, которые работают с общими данными на хосте: генератор создаёт CSV-файл с данными, а аналитик строит HTML-отчёт со статистикой. Проект демонстрирует работу с Docker-контейнерами, монтирование томов и автоматизацию через Bash-скрипты.

### Основные возможности

- Генерация CSV-файлов с произвольными данными (автомобили, товары, сотрудники и т.д.)
- Автоматический анализ данных и создание HTML-отчёта со статистикой
- Монтирование общих томов между контейнерами и хостом
- Управление проектом через единый Bash-скрипт `run.sh`
- Поддержка локальной отладки без Docker
- Просмотр содержимого контейнеров изнутри

### Структура проекта
* `generator/generate.py` — Python-скрипт генерации CSV-данных
* `generator/Dockerfile` — Docker-образ для генератора
* `reporter/report.js` — JavaScript-скрипт анализа данных
* `reporter/package.json` — зависимости Node.js
* `reporter/Dockerfile` — Docker-образ для аналитика
* `run.sh` — главный скрипт управления проектом
* `data/` — папка для общих данных (CSV и HTML)
* `local_data/` — папка для локальной отладки генератора
* `.gitignore` — исключения для Git
* `README.md` — описание и инструкция по запуску проекта

### Установка и запуск проекта
1. Клонируйте репозиторий:
   Откройте терминал на MacOs или командную строку на Windows и введите следующую команду:
   ```bash
   git clone https://github.com/iberbikov-code/hw3-docker-data.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd hw3-docker-data
   ```
3. Убедитесь, что Docker Desktop запущен:
   ```bash
   docker --version
   ```
4. Соберите образ генератора:
   ```bash
   ./run.sh build_generator
   ```
5. Запустите генератор данных:
   ```bash
   ./run.sh run_generator
   ```
6. Соберите образ аналитика:
   ```bash
   ./run.sh build_reporter
   ```
7. Запустите аналитик для создания отчёта:
   ```bash
   ./run.sh run_reporter
   ```
8. Проверьте результат:
   ```bash
   ls -la data/
   ```
   В папке `data/` должны появиться файлы `data.csv` и `report.html`

### Основные команды
- `./run.sh build_generator` — собрать образ генератора
- `./run.sh run_generator` — запустить генератор (создаёт data/data.csv)
- `./run.sh build_reporter` — собрать образ аналитика
- `./run.sh run_reporter` — запустить аналитик (создаёт data/report.html)
- `./run.sh create_local_data` — локальная генерация данных (без Docker)
- `./run.sh structure` — показать структуру проекта
- `./run.sh clear_data` — очистить папку data/
- `./run.sh inside_generator` — показать содержимое /data из контейнера генератора
- `./run.sh inside_reporter` — показать содержимое /data из контейнера аналитика
